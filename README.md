# Souvenir Shop v2

Скелет проекта интернет-магазина сувениров с поддержкой B2C/B2B, PWA и современным стеком:

- Frontend: Nuxt 3, Vue 3, TypeScript, Pinia, TailwindCSS, PWA.
- Backend: NestJS (Node.js + TypeScript).
- БД: PostgreSQL.
- Кэш: Redis.

На старте изображения хранятся локально, в дальнейшем возможно подключение внешнего хранилища.

## Быстрый старт (dev)

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run start:dev
```
