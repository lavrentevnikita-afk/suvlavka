# Design System — Souvenir Shop

Эта дизайн‑система описывает визуальный язык проекта: типографику, цвета, сетку, поведение элементов и ключевые паттерны интерфейса.  
Стиль вдохновлён современными luxury‑брендами: минимализм, аккуратная геометрия, мягкие скругления и воздушные отступы.

---

## 1. Цветовая палитра

Основные цвета проекта:

- **Pure White** — `#ffffff`  
  Базовый фон, большие блоки, негативное пространство.

- **Cloud Gray** — `#ececec`  
  Карточки, подложки, разделители, вторичный фон.

- **Lavender Blue** — `#a4b4ff`  
  Мягкие акценты, теги, hover‑состояния, вторичные CTA.

- **Electric Indigo** — `#3659fa`  
  Основные CTA‑кнопки, важные ссылки, элементы навигации и выделения.

Дополнительно допускаются нейтральные серые оттенки для текста и границ.

---

## 2. Типографика

### Шрифты

- **Bebas Neue**
  - Используется для крупных заголовков и промо‑блоков.
  - Регистр преимущественно верхний (ALL CAPS).
  - Создаёт ощущение плаката/обложки.

- **Inter**
  - Основной рабочий шрифт интерфейса.
  - Текст, описания, подсказки, навигация, цены.
  - Используются веса: Regular, Medium, Semibold, Bold.

### Общие принципы

- Заголовки страниц, промо‑блоки, крупные сообщения → **Bebas Neue**.
- Карточки товара, навигация, цены, описания → **Inter**.
- Межстрочные интервалы чуть увеличены для ощущения воздуха.
- Максимум две размерные иерархии на одном экране (не перегружаем).

---

## 3. Сетка (Grid System)

Базовая сетка для десктопа:

- **Отступы слева и справа**: `50 px`
- **Ширина колонки**: `97 px`
- **Межколоночный интервал (gutter)**: `16 px`

Сетка используется для:

- формирования карточек в каталоге;
- выстраивания галерей и промо‑баннеров;
- позиционирования контента на страницах товара.

На мобильных устройствах сетка упрощается, но основные пропорции сохраниются:

- фиксированные боковые отступы,
- единый ритм по вертикали,
- крупные кликабельные элементы.

---

## 4. Скругления (Border Radius)

Скругления — важный элемент общей визуальной идентичности.

Рекомендуемые значения:

- **24 px** — карточки товаров, промо‑баннеры, крупные контейнеры.
- **12 px** — основные кнопки и интерактивные элементы.
- **8 px** — инпуты, теги, мелкие кнопки.

Жёстких острых углов следует избегать — почти все элементы интерфейса должны иметь мягкое закругление.

---

## 5. Паттерны интерфейса

### 5.1. Промо‑карточки (баннеры)

- Фон: плавный градиент или глубокий тёмный цвет.
- Верхняя зона: круглые иконки действий/ссылок.
- Центральная зона: крупный заголовок Bebas Neue (white).
- Нижняя часть: теги с мягкими скруглениями (Inter, маленький размер).

### 5.2. Карточка товара в списке

- Фон: **Pure White** или **Cloud Gray**.
- Скругления: крупные.
- Верх: изображение товара с достаточным отступом от краёв.
- Правый верхний угол: иконка «избранное» в виде пустого сердечка в круге.
- Низ: название товара (Inter Medium), цена (Inter Semibold), опциональные цветовые точки/варианты.

### 5.3. Навигация (header)

- Горизонтальное меню на светлом или градиентном фоне.
- Активный пункт:
  - подчёркнут тонкой линией,
  - цветом может подсвечиваться Electric Indigo.
- Без лишних бордеров и коробок — текстовый, лёгкий, «невесомый».

### 5.4. Футер

- Три логических колонки:
  - Catalog
  - Company
  - Support
- Справа — круглые кнопки соцсетей с обводкой.
- Много свободного пространства, светлый фон.

### 5.5. Страница товара

- Слева: галерея продуктов (основное изображение + дополнительные).
- Справа: 
  - бренд и название,
  - рейтинг,
  - цена,
  - выбор цвета/размера,
  - контролы количества,
  - кнопка Add to cart (Electric Indigo),
  - кнопка избранного (outline).
- Внизу: вкладки/аккордионы Details, Delivery и т.п.
- Ниже по странице: блок Similar Products в виде аккуратной сетки карточек.

## 6. TailwindCSS и токены

В Tailwind‑конфиг добавляются дизайн‑токены:

```js
// tailwind.config.ts (фрагмент)
export default {
  theme: {
    extend: {
      colors: {
        pure: "#ffffff",
        cloud: "#ececec",
        lavender: "#a4b4ff",
        indigo: "#3659fa",
      },
      fontFamily: {
        sans: ["Inter", "system-ui", "sans-serif"],
        display: ["Bebas Neue", "system-ui", "sans-serif"],
      },
      borderRadius: {
        xl: "24px",
        lg: "12px",
        md: "8px",
      },
      container: {
        center: true,
        padding: "50px"
      }
    }
  }
}
```

Компоненты должны использовать эти токены вместо «магических» значений.

---

## 7. Анимации и микровзаимодействия

- Лёгкие переходы (200–250 ms, easing `ease-out` или `ease-in-out`).
- Hover‑состояния плавные, без резкого изменения цвета.
- Нажатие на кнопку:
  - лёгкое затемнение/сжатие,
  - без агрессивных «вспышек».
- Для карточек — мягкий подъём/тень при наведении.

---

Эта дизайн‑система будет расширяться по мере появления новых паттернов и компонентов.  
Основной принцип: минимализм, аккуратность и фокус на продукте.
